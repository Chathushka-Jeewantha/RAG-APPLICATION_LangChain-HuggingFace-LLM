
# RAG-APPLICATION: LangChain & Hugging Face LLM

## Overview
This repository contains a **Retrieval-Augmented Generation (RAG) application** built using **LangChain** and **Hugging Face LLMs**. The application enhances text generation by integrating an external knowledge base, enabling it to retrieve and generate more accurate and context-aware responses.

## Features
- **Document Ingestion:** Supports various document formats for knowledge retrieval.
- **Vector Database Integration:** Uses FAISS for efficient document search.
- **LLM Integration:** Leverages Hugging Face's pre-trained models for response generation.
- **LangChain Pipelines:** Implements structured chains for retrieval and generation workflows.
- **FastAPI Server:** Provides an API interface for querying the RAG system.

## Tech Stack
- **Python**
- **LangChain**
- **Hugging Face Transformers**
- **FAISS / ChromaDB**
- **FastAPI**
